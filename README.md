# wxapp_store
商城小程序(微信)——功能齐全且强大的微信商城小程序。

## 一、功能列表<br>
* 基本商城功能(商品管理，快递，购物车，微信支付，充值，会员卡，优惠券)<br>
* 多商城功能<br>
* 积分功能<br>
* 三级分销功能<br>
* 股东分销功能<br>
* 限时抢购<br>
* 整点秒杀<br>
* 超值拼团<br>
* 好友砍价<br>
* 分销提现<br>
* 每日签到<br>
* 快递跟踪<br>
* 上门自提<br>
* 售后退款<br>
* 活跃用户跟踪<br><br>
